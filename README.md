# OCPS Learning Management System

Welcome to the Oxford College of Professional Studies (OCPS) LMS.

## How to Run
Simply open `index.html` in your web browser. No server installation is required as this prototype uses local storage for data persistence.

## Login Credentials

### Administrator
- **Email:** `admin@ocps.edu`
- **Password:** `admin123`
- **Permissions:** Manage students (add/remove), Upload recordings and assignments.

### Student
- **Email:** `student@ocps.edu`
- **Password:** `student123`
- **Permissions:** View enrolled courses, Watch recordings, Access assignments.

## Features
- **Responsive Design:** Works on desktop and mobile.
- **Navy Blue Theme:** Professional color palette.
- **Data Persistence:** Changes (new students, uploads) are saved in your browser's local storage.
